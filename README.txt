Juego de Memoria Visual - Java Swing
=====================================

Cómo compilar:

1. Ve a la carpeta 'src'.
2. Compila todos los archivos Java:
   javac *.java

Cómo ejecutar:

3. Ejecuta la clase principal:
   java MemoriaGame

Notas:
- Necesitas que el directorio 'assets/' esté en la misma ruta de ejecución.
- Imágenes básicas generadas automáticamente.
